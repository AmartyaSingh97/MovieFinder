# MovieFinder
Infinite scrolling movie list jetpack compose app using TMDB API and add to favorites option with language filter. 

## Tech used: -
- Jetpack compose
- Pagination3
- Dagger Hilt 
- RoomDB
- Retrofit
- MVVM
- Material3 UI
